# discord-bot

### Requirements

- Ruby
- Rake

### Dependencies

- [discordrb 3.2.1](https://www.rubydoc.info/gems/discordrb/3.2.1/)
- [rspotify](https://www.rubydoc.info/github/guilhermesad/rspotify/)

### Installation

```
bundle install
rake run
```

### API Keys

- ``discord_client_id``
- ``discord_token``
- ``spotify_client_id``
- ``spotify_client_secret``
