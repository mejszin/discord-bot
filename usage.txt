# Projects (in-development)

> List active community projects
~projects
> Add a new project (title cannot contain spaces)
~project-add <url> <title> <description>
> Disable your project (you can re-enable later)
~project-disable <title>
> Enable your project
~project-enable <title>
> Join a project
~project-join <title>             
> Leave a project
~project-leave <title>

# Project tasks (in-development)

> List all current tasks for a project
~project-tasks <title>
> Display progress bars for task categories
~project-task-progress <title>
> Add a new task category (category cannot contain spaces)
~project-task-category-add <title> <category>
> Remove an existing task category
~project-task-category-remove <title> <category>
> Remove all tasks from category
~project-task-category-clear <title> <category>
> Add a new task to a category
~project-task-add <title> <category> <description>
> Remove a task
~project-task-remove <title> <category> <index>
> Overwrite a task
~project-task-overwrite <title> <category> <index> <description>
> Complete a task
~project-task-complete <title> <category> <index>

# Spotify

> Show a track message
~spotify <query>
> Show a track message with more information
~spotify-info <query>
> List of available genres for suggestions
~spotify-genres
> Search for 3 random suggestions by genres
~spotify-suggest <genres>
> Show all the emojis used to show energy level
~spotify-gamut
